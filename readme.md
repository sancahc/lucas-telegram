# lucas-telegram

## Usage

```sh
$ yarn
$ source .env
$ yarn dev
```

## Deployment

This bot can be deployed to [now](https://zeit.co/now) by Zeit.
Assuming you've got `now` installed and set up:

```sh
$ now -e BOT_TOKEN='123:......' sancahs/lucas-telegram
```

Alternative, deploy right now without even leaving the browser:

[![Deploy to now](https://deploy.now.sh/static/button.svg)](https://deploy.now.sh/?repo=https://github.com/sancahs/lucas-telegram&env=BOT_TOKEN)
